samePage

An app for group discussions. Users can simply name their conversation, create a password, and share these credentials with whomever they'd like to join in, eliminating the need to keep track of lengthy, randomly-generated URLs.

This is a fullstack React app with JWT authentication.

Inside Voices

Bridging the gap between New Yorkers who don't have social media accounts and the bounty of circulated resources that they might otherwise miss out on. Provides users with a daily journal, guided and instrumental meditations, and a list of free food fridges, local aid efforts, and low cost mental health care.

I constructed this as a full stack app, using React on the client side and building a RESTful API and authentication system on the server side. I plan to expand this project.

Tech Used: PostgreSQL, Node, Express, React, HTML5, CSS
